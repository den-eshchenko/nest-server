## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
 node v14.17.6
```bash
$ npm install
```
install Claster mongo - https://cloud.mongodb.com/

add .env -

JWT_KEY,
USER_LOGIN,
USER_PASSWORD,
PORT,
MONGODB_HOST,
MONGODB_USER,
MONGODB_PASSWORD,
MONGODB_PROTOCOL


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
